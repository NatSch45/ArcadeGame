## Controls (keyboard)

### Player 1

- **D** : go right
- **Q** : go left
- **Z** : jump
- **S** : stoop
- **SPACE** : punch
- **LEFT ALT** : kick
- **LEFT CONTROL** : hadouken

### Player 2

- **RIGHT ARROW** : go right
- **LEFT ARROW** : go left
- **UP ARROW** : jump
- **DOWN ARROW** : stoop
- **!** : punch
- **RIGHT CONTROL** : kick
- **RIGHT SHIFT** : hadouken
